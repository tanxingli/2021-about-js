2、浏览器默认数字都是10进制

3、以00,0o开头默认八进制

4、整数十进制转为2进制

```

(34).toString(2) // 100010
=>34/2  17...0
=>17/2  8...1
=>8/2   4...0
=>4/2   2...0
=>2/2   1...0
=>1/2   0...1

```
5、小数十进制转为2进制

